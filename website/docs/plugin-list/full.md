### com.msgbyte.github

github 集成

目前支持接受来自`Github webhooks`的推送

### com.msgbyte.linkmeta

Url元数据展示

用于获取消息中网址的基本信息，如标题/概述/缩略图

支持媒体路径，直接显示媒体播放器

特殊支持bilibili，自动加载b站iframe播放器

### com.msgbyte.simplenotify

简易机器人，用于简易推送消息到群组频道

### com.msgbyte.tasks

任务管理插件，在个人面板中增加TODO面板，用于个人待办事项的管理


### com.msgbyte.meeting

`WIP`

tailchat视频会议与语音频道支持
