# 概述

`Tailchat` 是一款插件化易拓展的开源 IM 应用。可拓展架构赋予 `Tailchat` 无限可能性。

前端微内核架构 + 后端微服务架构 使得`Tailchat`能够驾驭任何定制化/私有化的场景

## 特性

- 完整的即时通讯基础能力
- 插件化架构的赋予的自由拓展能力
- 微服务架构赋予的水平拓展能力

## 技术栈

- 前端
  - `React`
  - `Redux`
  - `MiniStar`
  - `tailwindcss`
  - `iconify`
- 后端: 
  - `Nodejs`
  - `Socket.io`
  - `koa`
  - `moleculer`
