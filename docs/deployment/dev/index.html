开发环境

对于开发环境的搭建，tailchat 提供了非常简单快捷的方式

后端#

使用Docker快速搭建依赖环境#

mongodb

docker run -d --name mongo -p 27017:27017 mongo:4

redis

docker run -d --name redis -p 6379:6379 redis

minio

docker run -d \  -p 19000:9000 \  -p 19001:9001 \  --name minio \  -e "MINIO_ROOT_USER=tailchat" \  -e "MINIO_ROOT_PASSWORD=com.msgbyte.tailchat" \  minio/minio server /data --console-address ":9001"

启动开发服务器#

cp .env.example .envvim .env

编辑.env的配置为自己相关的上下文

yarn install # 安装环境变量yarn dev # 启动开发服务器

前端#

cd webyarn install # 安装依赖yarn plugins:all # 编译插件yarn dev # 进入开发模式
